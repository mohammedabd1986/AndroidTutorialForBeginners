# تطبيق مخزن الباركود (Flutter)

تطبيق Flutter يعمل على iOS وAndroid لقراءة الباركود، جلب البيانات من Google Sheets مرة واحدة فقط، ثم العمل بشكل كامل دون اتصال عبر التخزين المحلي (Hive).

## المزايا الرئيسية
- مسح باركود بالكاميرا (MobileScanner).
- مزامنة بيانات Google Sheets عبر نقطة نهاية Google Apps Script وإبقائها في Hive.
- دعم كامل للغة العربية واتجاه RTL.
- بحث يدوي أو عبر الكاميرا مع رسالة عربية واضحة عند عدم العثور على الباركود.
- شاشة مزامنة تعرض عدد العناصر المخزنة وإشعارات النجاح أو الفشل.

## بدء الاستخدام
1. ضع كود Google Apps Script من `google_apps_script.js` في مشروع Google Apps Script متصل بجدولك، وانشره كتطبيق ويب عام أو مقيّد للمستخدمين المصرح لهم.
2. انسخ رابط النشر (ينتهي بـ `/exec`) والصقه في حقل "رابط Google Apps Script" داخل شاشة المزامنة في التطبيق.
3. اضغط "تحديث البيانات" لتحميل جميع الصفوف مرة واحدة وتخزينها محليًا.
4. استخدم تبويب "المسح والبحث" للبحث اليدوي أو مسح الباركود. في حالة عدم العثور على الباركود ستظهر الرسالة: `الباركود غير موجود في قاعدة البيانات`.

## تشغيل التطبيق محليًا
```sh
flutter pub get
flutter run
```

## الهيكل
- `lib/main.dart`: نقطة الدخول، تهيئة Hive وMaterialApp بالعربية.
- `lib/data/`: خدمات التخزين المحلي وطلب Google Sheets.
- `lib/screens/`: شاشات المزامنة والمسح والواجهة الرئيسية.
- `lib/widgets/`: عناصر العرض مثل بطاقة النتيجة.
- `google_apps_script.js`: كود واجهة Google Sheets.
